SnapLogic Pipeline Explorer

Adds a sidebar showing open SnapLogic pipelines with grouping, search, and collapsible sections

Settings

URL Pattern Configuration:
Set the URL pattern where the extension will be active. Use * as wildcards.
Examples: https://*.snaplogic.com/*, https://mycompany.snaplogic.com/*

Note: You may need to reload the extension or refresh pages for changes to take effect.

Emoji Customization

Default Emoji Set:
Customize the emojis that appear in the emoji picker. Separate emojis with spaces.
You can use any Unicode emojis or characters.

Interface Behavior

Dock Position:
Choose where the sidebar should dock when using the dock button.

Undock Behavior:
Choose whether the sidebar should be expanded or minimized when restored from dock position.

Search Highlighting

Customize Search Highlights:
Configure how search results are highlighted in the pipeline canvas.
